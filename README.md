# Convolutional Code 編碼測試
## 使用環境
* Python3.8 以上
* macOS / Linux / Windows 
## 使用教學

1. 使用pipenv管理套件
    
    ```
    pip intall pipenv 
    ```
2. 安裝依賴套件

    ```
    pipenv install
    ```
3. 開始使用
    ```
    pipenv shell
    ```
    ```
    python -u "./main.py"  
    ```
    

## 可動參數 
  1. g0、g1 
  
  2. BitsNum
